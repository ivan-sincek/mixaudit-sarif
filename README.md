# MixAudit SARIF

Convert MixAudit's JSON formatted results to SARIF format.

Tested on Kali Linux v2021.4 (64-bit) and with GitHub Actions.

Made for educational purposes. I hope it will help!

## How to Run

Open your preferred console from [/src/](https://github.com/ivan-sincek/mixaudit-sarif/tree/master/src) and run the following Bash command:

```fundamental
python mixaudit_sarif.py
```

Check the workflow [here](https://github.com/ivan-sincek/mixaudit-sarif/blob/main/workflows/mixaudit-analysis.yml).
